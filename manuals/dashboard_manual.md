## Introduction
This manual is for **Dashboard** users who need to monitor dispatch activities, view reports, and track key performance indicators.

## Minimum Requirements
- A modern web browser (Chrome, Edge, or Firefox).
- Stable internet connection.

## Logging In
1. Open the provided application URL.
2. Enter your **Username** and **Password**.
3. Click **Login**.

> *If you do not have a login, contact the Digital Transformation Department.*

## Dashboard Overview
The dashboard displays:
- **Summary Tiles** – Quick view of total dispatches, pending approvals, and completed orders.
- **Recent Activities** – List of the latest actions performed in the system.
- **Charts** – Visual representation of dispatch trends over time.

---

## Using Dashboard Features
- **Filter Data** – Use the dropdowns to select date ranges, status, or user.
- **Export Report** – Click the **Export** button to download the current view as an Excel file.
- **Refresh** – Click the **Refresh** icon to update the data.

> **Note:** Changing any dashboard settings (e.g., default filters) requires assistance from the admin team.

---

## Handling Errors
| Symptom | Possible Cause | Suggested Action |
|---|---|---|
| Dashboard shows no data | No dispatches created yet | Verify that dispatches exist; try a broader date range. |
| Export fails | Browser popup blocked | Allow pop‑ups for the site or use a different browser. |
| Page is stuck loading | Network latency | Check internet connection; retry after a few minutes. |

If problems persist, see **Contact Information**.

---

## Frequently Asked Questions
**Q:** How do I change the date range?
**A:** Click the date picker at the top of the dashboard and select the desired range.

**Q:** Can I customize the charts?
**A:** Customization is limited to filter selection; for advanced changes, contact the admin team.
